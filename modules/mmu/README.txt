-= Mix Modules Up module =-

Summary
=========================


Requirements
=========================
MixItUP library should be installed to libraries/mixitup directory.

Installation
=========================
 * Install as usual, see http://drupal.org/node/895232 for further information.
 * Download MixItUp library from https://mixitup.kunkalabs.com/.
 * Extract the jquery.mixitup.min.js file to libraries/mixitup directory.

Notes
=========================
You can use `drush mmu-download` command for easy installation of the library.
